# images/

Place your portfolio images here. The filenames expected by index.html are:

## Profile Photo
- `profile.jpg`  → your profile photo (used in the hero card)

## Certification Images
- `cert-01.jpg`  → Full Stack Web Development certificate
- `cert-02.jpg`  → Web Development Fundamentals certificate
- `cert-03.jpg`  → JavaScript & React certificate
- `cert-04.jpg`  → Database Management certificate
- `cert-05.jpg`  → Software Engineering Basics certificate

## How to activate images in index.html

1. Copy your image files into this folder.
2. Open `index.html` and find the comment blocks like:
   <!--
     <img src="images/cert-01.jpg" alt="Certificate" class="cert-img-preview"/>
   -->
3. Remove the <!-- and --> to uncomment the image tag.
4. The certificate image will appear as a preview inside the card.

## Tips
- Recommended format: JPG or PNG
- Certificate images look best at roughly 800×600px or 16:9 ratio
- Profile photo should be square (at least 200×200px)
